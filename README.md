To-Do List Web App

A clean, responsive, and accessible To-Do List app built with HTML, Tailwind CSS, and JavaScript. Keep track of your tasks with a modern interface and persistent storage in localStorage.

Features

✅ Add, edit, and delete tasks

✅ Mark tasks as completed with a click

✅ Clear all tasks at once

✅ Keyboard-friendly with Enter key support

✅ Responsive design for mobile, tablet, and desktop

✅ Lightweight SVG icons for actions

Quick Start

1. Clone the repo:

2. git clone https://github.com/sepherd/todo.git

3. Open index.html in your browser

4. You can also use the live demo hosted on GitHub Pages: https://sepherd.github.io/todo/

Start managing your tasks!
